## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `kissology`).
2. Copy the contents of `kissology-site/` into the repo root.
3. Commit & push.
4. In **Settings → Pages**, set:
   - Source: `Deploy from a branch`
   - Branch: `main` / root (`/`)
5. Your site will be live at `https://<username>.github.io/<repo>/`

That's it — no build, no bundler, no config.
